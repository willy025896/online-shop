# Prompts

記錄所有使用者對 AI 的請求。

## 命名規則

`YYYY-MM-DD/XXX-short-description.md`

- `YYYY-MM-DD`: 日期資料夾
- `XXX`: 當日順序編號 (001, 002, 003...)
- `short-description`: 簡短描述

## 範例
- 2026-03-06/001-add-payment-feature.md 
- 2026-03-06/002-fix-cart-validation.md 
- 2026-03-06/003-refactor-order-service.md 
- 2026-03-07/001-create-admin-dashboard.md

## 檔案內容

每個 prompt 記錄應該包含:
- 使用者原始請求
- 任務類型 (Feature/Bug Fix/Refactor/Documentation)
- 預期結果
- 執行狀態

詳細格式請參考 `AI-SKILLS.md` 中的模板。

## 提示

- 每天的編號從 001 開始
- 描述要清楚簡潔
- 與對應的 changes 記錄使用相同編號
